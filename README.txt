== Description ==

Nfinite Dashboard is a WordPress admin dashboard plugin designed to streamline workflow, enhance productivity, and provide quick access to essential tools. It replaces the default WordPress dashboard with a fully customizable interface that organizes and consolidates tasks, projects, meetings, and client management all in one place.

### Features:
- Custom admin dashboard with task and project management.
- Manage meetings, clients, and notes directly from the dashboard.
- AJAX-powered inline editing for faster workflow.
- User-friendly interface designed for WordPress professionals.
- Customizable columns for better task and project visibility.
- Built-in client search and filtering tools.
- Performance-optimized admin scripts and styles.

== Installation ==

1. Upload the `nfinite-dashboard` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Nfinite Dashboard** in the admin menu to start managing tasks and projects.

== Frequently Asked Questions ==

= How do I enable inline editing for tasks? =

Inline editing is enabled by default for task status, priority, and due dates. Simply navigate to **Tasks** in the admin dashboard and update fields directly.

= Can I manage multiple projects and clients? =

Yes, Nfinite Dashboard supports managing multiple projects and clients within a structured workflow.

= Does the plugin work with custom post types? =

Yes, Nfinite Dashboard extends WordPress' custom post types, allowing seamless integration with tasks, projects, meetings, and notes.

= How do I filter completed tasks? =

A "View Completed Tasks" button is available in the **Tasks** section, allowing you to switch between active and completed tasks.
